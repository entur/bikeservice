# Bikes

Currently we offer data from Oslo Bysykkel, Bergen Bysykkel, Trondheim Bysykkel and Kolumbus Bysykkel. All of our data is served in GBFS format which is
a standardized data feed for bikesharing systems. More information about GBFS can be found at https://github.com/NABSA/gbfs.

## List the GBFS endpoint for all operators

All currently implemented operators and their respective gbfs endpoint will be listed under the root endpoint.

### HTTP Request

`GET https://api.entur.io/mobility/v1/bikes`

### Example response

```
{
    "operators": [
        {
            "oslobysykkel": "https://api.entur.io/mobility/v1/bikes/oslobysykkel/gbfs.json"
        },
        {
            "bergenbysykkel": "https://api.entur.io/mobility/v1/bikes/bergenbysykkel/gbfs.json"
        },
        {
            "trondheimbysykkel": "https://api.entur.io/mobility/v1/bikes/trondheimbysykkel/gbfs.json"
        },
        {
            "kolumbusbysykkel": "https://api.entur.io/mobility/v1/bikes/kolumbusbysykkel/gbfs.json"
        }
    ]
}
```

## GBFS information for an operator

Get the different GBFS endpoints for a specific operator.

### HTTP Request

`GET https://api.entur.io/mobility/v1/bikes/{operator}/gbfs.json`

Where `{operator}` is one of the above mentioned operators, i.e either `oslobysykkel`, `kolombusbysykkel`, `bergenbysykkel`, `trondheimbysykkel`.

### Example response

```
{
    "data": {
        "nb": {
            "feeds": [
                {
                    "name": "system_information",
                    "url": "https://api.entur.io/mobility/v1/bikes/oslobysykkel/system_information.json"
                },
                    {
                    "name": "station_information",
                    "url": "https://api.entur.io/mobility/v1/bikes/oslobysykkel/station_information.json"
                },
                {
                    "name": "station_status",
                    "url": "https://api.entur.io/mobility/v1/bikes/oslobysykkel/station_status.json"
                }]
            }
        },
    "last_updated": 1589365951,
    "ttl": 15
}
```

## System information

Get the information about the system for an operator.

### HTTP Request

`GET https://api.entur.io/mobility/v1/bikes/{operator}/system_information.json`

### Example response

```
{
    "data": {
        "system_id": "oslobysykkel",
        "language": "nb",
        "name": "Oslo Bysykkel",
        "operator": "UIP Oslo Bysykkel AS",
        "timezone": "Europe/Oslo",
        "phone_number": "+4791589700",
        "email": "post@oslobysykkel.no"
        },
    "last_updated": 1589366110,
    "ttl": 10
}
```
